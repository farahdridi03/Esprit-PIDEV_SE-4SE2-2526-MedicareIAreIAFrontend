🏥 MEDICARE AI

📖 Description

MEDICARE AI est une application web innovante dédiée à la gestion des services de santé.
Elle permet de centraliser les données médicales, de faciliter l’accès aux soins et d’optimiser la prise de décision médicale grâce à l’intégration de l’intelligence artificielle.

🚀 Fonctionnalités

👤 Gestion des utilisateurs : patients, médecins et administrateurs
📁 Dossier médical électronique sécurisé
📅 Prise de rendez-vous en ligne
🧪 Gestion des analyses de laboratoire
💊 Suivi des pharmacies et des médicaments
🏠 Services médicaux à domicile
🔔 Système de notifications intelligentes
💬 Forum de discussion & gestion des événements
🤖 Modules d’intelligence artificielle pour l’aide au diagnostic et recommandations

🛠️ Technologies

🎨 Frontend : Angular
⚙️ Backend : Java (Spring Boot)
🗄️ Base de données : MySQL
🔗 Architecture : API REST

⚙️ Installation

Cloner le projet :
git clone https://github.com/ton-username/medicare-ai.git
Accéder au dossier du projet :
cd medicare-ai
Installer les dépendances et lancer les serveurs (frontend & backend)

▶️ Utilisation

Créer un compte utilisateur
Se connecter à la plateforme
Accéder aux différents services médicaux disponibles
Gérer les rendez-vous, dossiers médicaux et interactions en toute simplicité
