🏥MEDICARE AI
📖 Description
MEDICARE AI est une application web de gestion des services de santé.
Elle permet de centraliser les données médicales et d’améliorer la qualité des soins grâce à l’intelligence artificielle.

🚀 Fonctionnalités
👤 Gestion des utilisateurs (patient, médecin, admin)
📁 Dossier médical
📅 Prise de rendez-vous
🧪 Analyses de laboratoire
💊 Gestion des pharmacies
🏠 Services à domicile
🔔 Notifications
💬 Forum et événements
🤖 Intelligence artificielle
🛠️ Technologies
Frontend : Angular
Backend : Java (Spring Boot)
Base de données : MySQL
API REST
⚙️ Installation
Cloner le projet :
git clone https://github.com/ton-username/medicare-ai.git
## ▶️ Utilisation
Créer un compte, se connecter et utiliser les services médicaux.
